# 🎂 Digital Love Letter: Perayaan Ulang Tahun Spesial

Sebuah proyek interaktif, *handcrafted*, dan *premium* yang dibangun khusus sebagai kado ulang tahun untuk seseorang yang istimewa. Website ini dirancang bukan sekadar sebagai halaman web, melainkan sebuah ruang digital untuk merayakan kenangan, harapan, dan kasih sayang.

---

## 🎨 Pengalaman Interaktif
Website ini dilengkapi dengan berbagai fitur premium untuk memberikan pengalaman yang tak terlupakan:

*   **Scrapbook Timeline**: Lembaran kenangan hidup yang menceritakan perjalanan kita.
*   **Draggable Polaroid Gallery**: Galeri foto interaktif yang bisa digeser dan disusun sesuka hati.
*   **Romantic Letter**: Surat cinta digital dengan animasi pembukaan amplop yang elegan.
*   **Virtual Gift Box**: Kado kejutan interaktif dengan efek *confetti*.
*   **Guestbook**: Ruang untuk menyimpan pesan-pesan harapan yang penuh makna.
*   **Music Visualizer**: Pemutar musik minimalis yang merespons irama lagu favorit.
*   **Dynamic Theme**: Perubahan suasana (Day/Night mode) otomatis sesuai waktu.

---

## 🛠️ Cara Mengatur (Setup)

Untuk mempersonalisasi kado ini, Anda hanya perlu mengganti file aset yang ada tanpa perlu mengubah kode pemrograman:

### 1. Mengganti Foto
Buka folder `assets/images/`. Ganti file foto yang ada dengan foto Anda sendiri. Pastikan nama file tetap sama agar website otomatis mengenalinya:
- `photo1.jpg`
- `photo2.jpg`
- `photo3.jpg`
- `photo4.jpg`
- `photo5.jpg`
*(Disarankan menggunakan foto rasio 3:4 agar tata letak sempurna).*

### 2. Mengganti Musik
Buka folder `assets/music/`. Ganti file `birthday-song.mp3` dengan lagu favorit kalian (pastikan nama file tetap sama).

## 🚀 Cara Menjalankan & Deploy

Proyek ini dibangun menggunakan **Vanilla HTML5, CSS3, dan JavaScript (ES6+)**. Tidak perlu *build tool*, *framework*, atau *database* yang rumit.

1.  **Local Testing**: Anda bisa langsung membuka `index.html` di browser (Chrome/Safari) untuk melihat hasilnya di komputer Anda.
2.  **Deploy ke GitHub Pages**:
    - Unggah seluruh folder ini ke Repository GitHub Anda.
    - Buka menu **Settings** > **Pages** di Repository tersebut.
    - Pilih branch **main** sebagai source.
    - Website Anda akan segera bisa diakses melalui link yang diberikan oleh GitHub.

## ⚙️ Teknologi yang Digunakan
*   **HTML5**: Semantik dan aksesibel.
*   **CSS3**: *Glassmorphism*, *Keyframe Animations*, *Responsive Design*.
*   **Vanilla JS**: Manipulasi DOM, *Particle System*, Audio API, dan *Physics-based interactions*.
*   **PWA Ready**: Terintegrasi dengan `manifest.json` agar bisa diinstal sebagai aplikasi di smartphone.
